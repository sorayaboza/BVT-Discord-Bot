# BVT-Discord-Bot
BVT's Discord Bot. Collects voice data of BVT interns on Discord calls.


Prerequisits: MySQL and Nodejs.

To run:
1. Type 'npm i' in terminal.
2. Run schema.sql to create the database.
3. Run the app.js file.
